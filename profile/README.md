## 🧬 Sobre NeusRecords
Somos una organización dedicada a la creación de software artesanal, centrada en la **estética visual**, el **rendimiento** y la **usabilidad**. Creemos que el código no solo debe funcionar, debe inspirar.

- 🛠️ Actualmente construyendo herramientas de diseño y plataformas de comunicación.
- 🎨 Enfoque en: UI/UX Minimalista, Glassmorphism y Mobile First.
- ⚡ Pasión por el código abierto y la colaboración global.
